T6 United Console.

T6 United Console is an external console for T6 which supports the following executables:

  • t6mp
  • t6zm
  • t6mpv43
  • t6mpv41

Usage

T6 United Console operates very similarly to all other external consoles. Single commands can be executed from the topmost input box, while multiple commands can be executed simultaneously in the large input box.

Features

T6 United Console currently supports:

  • Single command execution
  • Multiple command (config) execution
  • Config loading
  • Config saving
  • Consistent game compatibility updates
